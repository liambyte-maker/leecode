27 移除元素

```python
class Solution:
    def removeElement(self, nums: List[int], val: int) -> int:
        fast=0
        slow=0
        size=len(nums)
        while fast<size:   #不能等于，防止超过长度
            if nums[fast] !=val:
                nums[slow]=nums[fast]
                slow+=1
            fast+=1
        return slow
```
快慢指针，快指针一直向前走，当遇到不等于val的时候，用slow去储存nums
最后return nums里面不等于val的数量
