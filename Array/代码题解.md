27 移除元素

```python
class Solution:
    def removeElement(self, nums: List[int], val: int) -> int:
        fast=0
        slow=0
        size=len(nums)
        while fast<size:   #不能等于，防止超过长度
            if nums[fast] !=val:
                nums[slow]=nums[fast]
                slow+=1
            fast+=1
        return slow
```
快慢指针，快指针一直向前走，当遇到不等于val的时候，用slow去储存nums
最后return nums里面不等于val的数量


977 有序数组的平方


```python
class Solution:
    def sortedSquares(self, nums: List[int]) -> List[int]:
        l,r,i=0,len(nums)-1,len(nums)-1
        res=[0]*len(nums)
        while l<=r:
            if nums[l]**2<nums[r]**2:
                res[i]=nums[r]**2
                r-=1
            else:
                res[i]=nums[l]**2
                l+=1
            i-=1
        return res

```
因为本身nums就是非递减顺序，所以最大的一定是在最左边或最右边。
先给占位数组，然后比大小，如果左边小于右边，则右边的添加到新的数组里面，right index -=1,如果左边大于右边，则左边的添加到新的数组里，左边index+=1 从后往前添加
